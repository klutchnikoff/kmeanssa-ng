    Quick Start¶

    Get up and running with kmeanssa-ng in 5 minutes! This guide will walk you through your first clustering example.

    Basic Example¶

    Let’s cluster points on a simple quantum graph with two natural clusters:

    from kmeanssa_ng import generate_sbm, QGSimulatedAnnealing

# Step 1: Create a quantum graph with block structure
graph = generate_sbm(
    sizes=[30, 30],  # Two clusters of 30 nodes each
    p=[
        [0.8, 0.1],  # 80% intra-cluster edges, 10% inter-cluster
        [0.1, 0.8],
    ],  # 10% inter-cluster, 80% intra-cluster
)

# Step 2: Precompute distances (important for performance!)
graph.precomputing()

# Step 3: Sample random points on the graph
points = graph.sample_points(100)  # 100 points distributed on edges

print(graph)
print(f"Created graph with {graph.number_of_nodes()} nodes and {graph.number_of_edges()} edges")
print(f"Sampled {len(points)} points on the graph")

    QuantumGraph named 'stochastic_block_model' with 60 nodes and 770 edges
Created graph with 60 nodes and 770 edges
Sampled 100 points on the graph

    # Step 4: Run k-means clustering
sa = QGSimulatedAnnealing(
    observations=points,  # Correct parameter name
    k=2,  # Number of clusters
    lambda_param=1,  # Temperature parameter
    beta=1.0,  # Drift strength
    step_size=0.1,  # Step size for random walks
)

# Step 5: Get cluster centers
centers = sa.run(
    robust_prop=0.1,  # Use last 10% iterations for averaging
    initialization="kpp",  # k-means++ initialization
    algorithm_version="v1",  # Algorithm variant
)

print(f"Found {len(centers)} cluster centers")
print(f"Centers located at: {[str(c) for c in centers]}")

    Found 2 cluster centers
Centers located at: ["QGPoint on 'stochastic_block_model' edge (1, 28) at position 0.152", "QGPoint on 'stochastic_block_model' edge (44, 13) at position 0.064"]

    Understanding the Results¶

    The algorithm returns centers - a list of QGCenter objects representing the optimal cluster centers. Each center has:

    • edge: The edge where the center is located
    • position: Position on the edge (between 0 and 1)
    for i, center in enumerate(centers):
    print(f"Cluster {i+1}: {center}")

    Cluster 1: QGPoint on 'stochastic_block_model' edge (1, 28) at position 0.152
Cluster 2: QGPoint on 'stochastic_block_model' edge (44, 13) at position 0.064

    Customizing the Algorithm¶

    Algorithm Parameters¶

    # More exploration (higher temperature)
sa_explore = QGSimulatedAnnealing(points, k=2, lambda_param=2, beta=0.5)

# More exploitation (stronger drift)
sa_exploit = QGSimulatedAnnealing(points, k=2, lambda_param=0.5, beta=2.0)

# Smaller steps (more precise)
sa_precise = QGSimulatedAnnealing(points, k=2, step_size=0.05)

print("Parameter variations created successfully")

    Parameter variations created successfully

    Different Graph Types¶

    Stochastic Block Model¶

    # Custom block probabilities
graph_3clusters = generate_sbm(
    sizes=[20, 30, 25], 
    p=[[0.9, 0.1, 0.05],
       [0.1, 0.8, 0.15], 
       [0.05, 0.15, 0.85]]
)
graph_3clusters.precomputing()
print(f"3-cluster graph: {graph_3clusters.number_of_nodes()} nodes")

    3-cluster graph: 75 nodes

    From NetworkX¶

    import networkx as nx
from kmeanssa_ng import as_quantum_graph

# Use any NetworkX graph
G = nx.karate_club_graph()
quantum_karate = as_quantum_graph(G, edge_length=1.0)
quantum_karate.precomputing()
print(f"Karate club as quantum graph: {quantum_karate.number_of_nodes()} nodes")

    Karate club as quantum graph: 34 nodes

    Performance Tips¶

    Speed Up Your Clustering:

    1. Always call precomputing() - This caches shortest paths
    2. Use appropriate robust_prop - 0.1 (10%) is usually sufficient
    3. Start with k-means++ - Set initialization="kpp"
    4. Tune parameters gradually - Start with defaults, then adjust
    # Fast setup for large graphs
graph.precomputing()  # Essential!
sa_fast = QGSimulatedAnnealing(points, k=3, step_size=0.2)  # Larger steps
centers_fast = sa_fast.run(robust_prop=0.05, initialization="kpp")  # Less robustification
print(centers_fast)
print(f"Fast clustering found {len(centers_fast)} centers")

    [QGCenter(edge=(1, 3), position=-0.303), QGCenter(edge=(44, 48), position=0.198), QGCenter(edge=(7, 29), position=0.446)]
Fast clustering found 3 centers
